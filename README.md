# A1 Mobile Sales & Service — Static Website

This is a simple, mobile-first static website for A1 Mobile Sales & Service (Lebbaikudikkadu).

Files:

- `index.html` — main site
- `css/styles.css` — styles
- `js/script.js` — client-side form handling and WhatsApp integration

Configuration:

- Open `js/script.js` and set `WHATSAPP_NUMBER` to the shop WhatsApp phone number in international format without the `+` sign. Example for +91 98765 43210 use `919876543210`.
- Update the phone displayed in the contact section in `index.html` if needed.

Behavior:

- All enquiry forms open WhatsApp with a pre-filled message. No data is stored on the site.
- Payment: Cash On Delivery only (no online payments).
- Free home pickup available only inside Lebbaikudikkadu (as described in the site).

How to run:

Open `index.html` in a browser on mobile or desktop. On mobile the WhatsApp links will open the WhatsApp app; on desktop they open WhatsApp Web.
